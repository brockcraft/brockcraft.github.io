Reviews API

Retrieve a paginated list of application reviews.

Endpoint

GET /reviews

Query Parameters

Parameter Type Required Description
offset integer No Number of records to skip. Default: 0
limit integer No Number of records to return. Default: 10
app string No Filter by app name
category string No Filter by category
rating integer No Filter by rating (1–5)
verified boolean No Filter by verified purchase status

Response

{
  "total": 500,
  "returned": 10,
  "offset": 0,
  "limit": 10,
  "reviews": [ ... ]
}

Top-Level Fields

Field Type Description
total integer Total number of matching reviews
returned integer Number of reviews returned
offset integer Offset used for the request
limit integer Limit used for the request
reviews array List of review objects

Review Object

{
  "id": 1,
  "app": "Fieldkit",
  "category": "field research",
  "rating": 1,
  "review": "Auto-transcription accuracy...",
  "date": "2023-03-31",
  "helpful_votes": 37,
  "verified_purchase": true
}

Fields

Field Type Description
id integer Unique review ID
app string Application name
category string Application category
rating integer Rating (1–5)
review string Review text
date string Date in YYYY-MM-DD format
helpful_votes integer Number of helpful votes
verified_purchase boolean Whether the review is from a verified user

Pagination

Use offset and limit to page through results.

GET /reviews?offset=10&limit=10

Continue paging until offset + returned >= total.

Example Request

GET /reviews?limit=10&offset=0

Example Response

{
  "total": 500,
